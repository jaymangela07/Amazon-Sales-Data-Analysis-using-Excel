# 📊 Amazon Sales Data Analysis – Excel Project

---

## 📌 Overview
This project analyzes Amazon sales data using **Microsoft Excel**.  
The dataset includes order details such as Order ID, Status, Fulfilment Type, Product Category, Quantity, and Sales Amount.

The project focuses on transforming raw sales data into meaningful insights using Excel tools and analytical techniques.

---

## 🎯 Objective
The main objective of this project is to:

- Analyze overall sales performance  
- Understand customer purchasing trends  
- Evaluate order status distribution  
- Identify high-performing product categories  
- Generate actionable business insights  

---

## 🛠 Tools Used

- Microsoft Excel  
- Pivot Tables  
- Charts & Graphs  
- Conditional Formatting  
- Excel Formulas (SUM, COUNTIF, SUMIFS, etc.)  

---

## 📊 Analysis Performed

✔️ Total Sales Calculation  
✔️ Order Status Analysis  
✔️ Category-wise Sales Analysis  
✔️ State-wise Revenue Analysis  
✔️ Fulfilment Type Comparison  
✔️ B2B vs B2C Sales Analysis  

---

## 📈 Key Insights

- Shipped orders dominate total transactions  
- Revenue loss observed due to cancelled orders  
- Specific product categories contribute significantly to total sales  
- Certain states generate higher revenue compared to others  

---

## ✅ Conclusion

The analysis provides valuable insights into sales performance and customer distribution patterns.  

Businesses can use these insights to:
- Improve inventory management  
- Reduce order cancellations  
- Target high-performing regions  
- Increase overall profitability
